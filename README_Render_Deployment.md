## 部署步驟

請按照以下步驟將您的 LINE 聊天機器人部署到 Render.com：

### 步驟 1: 上傳程式碼到 GitHub

將 `aka_chatbot.py`, `app.py`, `requirements.txt`, `Procfile`, `runtime.txt` 這五個檔案上傳到您的 GitHub 儲存庫中。建議將它們放在儲存庫的根目錄下。

### 步驟 2: 在 Render.com 建立新的 Web Service

1.  登入您的 Render.com 帳號。
2.  點擊儀表板上的「New +」按鈕，然後選擇「Web Service」。
3.  選擇您剛才上傳程式碼的 GitHub 儲存庫，並授權 Render.com 存取。
4.  配置您的 Web Service 設定：
    *   **Name**：為您的服務取一個名稱，例如 `line-chatbot-aka`。
    *   **Region**：選擇一個離您的用戶較近的區域。
    *   **Branch**：選擇您要部署的分支，通常是 `main` 或 `master`。
    *   **Root Directory**：如果您的程式碼在儲存庫的根目錄，請留空。否則，請填寫程式碼所在的子目錄名稱 (例如 `line_chatbot_aka`)。
    *   **Runtime**：Render.com 會自動偵測 `runtime.txt` 中的 Python 版本，如果沒有，請手動選擇 `Python 3.11.0`。
    *   **Build Command**：`pip install -r requirements.txt`
    *   **Start Command**：`gunicorn app:app --workers=1 --threads=8 --timeout=0 --bind 0.0.0.0:$PORT`
